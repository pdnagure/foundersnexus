# FounderNexus Logo Variations

## Files
- `foundernexus-v1-nexus-link-horizontal.svg`: Primary horizontal corporate mark.
- `foundernexus-v2-leadership-peak-stacked.svg`: Stacked badge with leadership/innovation motif.
- `foundernexus-v3-collab-orbit-emblem.svg`: Icon-only emblem for avatar/social/app use.
- `foundernexus-v4-corporate-bridge-wordmark.svg`: Structured corporate wordmark.
- `foundernexus-v5-conversation-hub.svg`: Meetup-first concept with communication symbolism.
- `foundernexus-v6-monochrome-print.svg`: Single-ink print-safe version.

## Core Palette
- Deep Navy: `#13253A`
- Corporate Blue: `#0F3D66`
- Growth Teal: `#11A77A`
- Warm Accent: `#FF7A59`
- Neutral Slate: `#4B6279`

## Recommended Usage
- Website header/footer: V1 or V4.
- Event banners/posters: V2 or V5.
- Social profile/favicon/app icon: V3.
- Letterheads/stamps/limited-color print: V6.

## Notes
- All files are vector SVG and scalable for digital and print.

  - dark-bg inverse
  - light-bg default
